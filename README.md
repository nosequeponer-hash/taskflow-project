# TaskFlow 📋

Aplicación web de gestión de tareas desarrollada como proyecto de prácticas.

## 🌐 Demo en vivo

👉 [https://taskflow-project-s1vw.vercel.app/](https://taskflow-project-s1vw.vercel.app/)

## ✨ Funcionalidades

- ✅ Añadir tareas nuevas escribiendo en el input y pulsando "Añadir" o Enter
- ✅ Marcar tareas como completadas haciendo clic en la tarjeta
- ✅ Eliminar tareas con el botón ✕
- ✅ Las tareas se guardan automáticamente y no se pierden al recargar la página
- ✅ Buscador en tiempo real para filtrar tareas por texto
- ✅ Diseño responsive (adaptado a móvil y escritorio)

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica con `<header>`, `<main>`, `<aside>` y `<section>`
- **CSS3** — Variables CSS, Flexbox, Media Queries y transiciones
- **JavaScript** — Manipulación del DOM y persistencia con LocalStorage
- **Vercel** — Despliegue y hosting

## 🚀 Cómo ejecutarlo en local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/nosequeponer-hash/taskflow-project.git
   ```
2. Abre la carpeta del proyecto
3. Abre `index.html` en tu navegador

No necesita instalación ni dependencias.
